#Подсчет калорий и БЖУ продуктов
Данный сервис позволяет пользователям получать данные о калориях, белках, жирах и углеводах (КБЖУ) продуктов с помощью стороннего API. Пользователи могут добавлять продукты в свои приемы пищи и отслеживать свое питание за день.
## Возможности
- Получение данных о КБЖУ продуктов с помощью стороннего API.
- Добавление продуктов в приемы пищи (завтрак, обед, ужин).
- Отображение истории приемов пищи и подсчет общей калорийности и общего количества БЖУ за день.
